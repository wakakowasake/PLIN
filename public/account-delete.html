PLIN 계정 관리

웹에서 계정 삭제 요청

이 페이지는 Google Play 스토어 등록정보에 표시되는 PLIN 계정 및 관련 데이터 삭제 요청 링크입니다. 앱에 접근할 수 없을 때도 같은 삭제 요청을 접수할 수 있어요.

앱 이름 PLIN(플린)
개발자 우지형
계정 삭제 URL https://plin.ink/account-delete
현재 로그인 상태

로그인된 계정을 확인하고 있어요.

계정 삭제 요청 단계

  1. 삭제하려는 PLIN 계정과 같은 로그인 방식(구글, 카카오, 네이버, 애플)으로 로그인합니다.
  2. 현재 로그인 상태에 표시된 계정이 삭제할 계정인지 확인합니다.
  3. 계정 삭제 요청 버튼을 누르고 확인 창에서 요청을 완료합니다.
  4. 요청이 완료되면 즉시 로그아웃되고, 관계 법령상 보관 의무가 있는 정보를 제외한 계정과 관련 데이터가 삭제됩니다.

삭제되는 데이터와 처리 기간

계정 삭제 요청이 완료되면 관계 법령상 보관 의무가 있는 정보를 제외하고 아래 데이터가 삭제됩니다.

  • Firebase 인증 계정, 사용자 프로필, 연결된 로그인 제공자 정보 및 토큰
  • 나만 남은 여행 일정, 삭제한 여행 보관함, 날짜별 타임라인, 장소, 메모, 가계부, 공유 링크 데이터
  • 공유 멤버가 남아 있는 여행에서는 내 멤버 정보가 삭제되고, 내가 소유자라면 소유권이 남은 멤버에게 자동으로 넘어갑니다.
  • 커뮤니티 게시글, 댓글, 좋아요, 신고·차단 관련 사용자 활동 기록
  • 푸시 알림 설치 정보와 알림 수신 상태
  • 계정 삭제 요청 처리에 필요한 계정 상태와 요청 시각
  • 프로필 사진, 일정 대표 사진, 추억 사진, 첨부 이미지·PDF 등 업로드 파일

구독 해지 안내

PLIN Plus 구독은 계정 삭제 요청만으로 자동 해지되지 않습니다.

  • App Store 또는 Google Play 구독 관리 화면에서 별도로 해지해야 합니다.
  • 앱 삭제, 로그아웃, 계정 삭제만으로는 다음 자동 갱신이 중단되지 않을 수 있습니다.
  • 해지와 환불은 사용한 앱마켓의 정책과 절차를 따릅니다.

보관될 수 있는 데이터와 추가 보관 기간

  • 삭제 요청 처리에 필요한 계정 상태와 요청 시각은 삭제 처리 중에만 사용되고 삭제 완료 시 함께 삭제됩니다.
  • 관계 법령상 보관 의무가 있는 정보가 발생한 경우에는 해당 법령에서 정한 기간 동안 분리 보관될 수 있습니다.
  • 유료서비스 거래, 청약철회, 결제 검증, 공급, 환불 및 분쟁 처리와 관련한 기록은 전자상거래 등 관계 법령에서 정한 기간 동안 분리 보관될 수 있습니다.
  • 서비스 보안, 오류 진단, 부정 이용 방지를 위한 서버 접속·오류 로그는 개인을 직접 식별하기 어렵게 최소화하여 최대 30일 동안 보관될 수 있습니다.
  • 사용자의 브라우저 또는 기기에 남은 캐시, 다운로드 파일, 스크린샷 등은 PLIN 서버 삭제와 별도로 사용자가 직접 정리해야 할 수 있습니다.
