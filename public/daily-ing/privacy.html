Daily-ing 개인정보처리방침

마지막 수정일 2026년 5월 31일

플린(PLIN)은 Daily-ing 사용자의 시간표와 계정 정보를 필요한 범위에서만 처리합니다.

Daily-ing은 로컬 저장을 기본으로 하며, 사용자가 로그인과 백업을 선택한 경우에만 서버에 일정 백업을 저장합니다.

1조. 개인정보의 처리 목적

  • 계정 인증: Google 또는 Apple 로그인, 사용자 식별, 계정 상태 관리
  • 일정 관리: 주간 시간표, 일회성 약속, 장소, 메모, 색상, 알림 설정 저장
  • 백업: 사용자가 로그인한 경우 시간표와 약속 데이터를 서버에 백업하고 복원
  • 알림: 일정 시작 알림, 진행 중 상시 표시, Live Activity 종료 처리
  • 품질 개선: 오류 진단, 앱 이용 현황 분석, 기능 안정화

2조. 처리하는 개인정보 항목

계정 정보: Firebase 사용자 식별값, 이메일, 표시 이름, 프로필 이미지 URL, 로그인 제공자

일정 정보: 제목, 요일, 시작·종료 시간, 장소, 강의실, 메모, 색상, 알림 및 표시 정책

기기 기능 정보: 알림 권한 상태, Live Activity push token, APNs 환경, 앱 버전, 기기 캘린더에서 사용자가 가져온 일정

분석 정보: Firebase Analytics가 제공하는 앱 실행, 오류, 이벤트 정보

3조. 외부 서비스 이용

Daily-ing은 인증, 데이터 저장, 분석, 알림 처리를 위해 Firebase Authentication, Cloud Firestore, Firebase Analytics, Firebase Cloud Functions를 사용할 수 있습니다.

iOS Live Activity와 푸시 관련 처리는 Apple APNs를, Android 알림은 기기 운영체제의 알림 기능을 사용합니다.

4조. 보유 및 삭제

로컬 일정 데이터는 사용자의 기기에 저장되며, 앱 삭제 또는 데이터 초기화 시 함께 삭제될 수 있습니다.

서버 백업 데이터는 사용자가 삭제하거나 계정을 삭제할 때까지 보관합니다. 계정 삭제 시 서버에 저장된 백업, Live Activity 서버 기록, Firebase 계정이 삭제됩니다.

관계 법령상 보관이 필요한 정보가 발생하는 경우 해당 기간 동안 분리 보관할 수 있습니다.

5조. 이용자 권리

사용자는 앱 설정에서 백업 삭제, 로그아웃, 계정 삭제를 요청할 수 있습니다.

개인정보 열람, 정정, 삭제, 처리정지 요청은 contact@plin.ink 로 문의할 수 있습니다.
