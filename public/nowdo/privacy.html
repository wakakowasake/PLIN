지금해 개인정보처리방침

시행일 2026년 7월 13일

플린(PLIN)은 지금해 사용자의 정보를 필요한 범위에서만 처리합니다.

계정 연결은 선택 사항이며, 루틴과 기록은 계정 연결 여부와 관계없이 사용자의 iPhone에 저장합니다.

1조. 처리 목적

  • 루틴 실행: 예약 시각, 반복 요일, 집중 시간에 따라 알람과 카운트다운 제공
  • 방해 차단: 사용자가 선택한 앱과 웹사이트를 집중 시간 동안 차단
  • 완료 기록: 루틴 완료 여부와 사용자가 선택적으로 촬영한 인증 사진 표시
  • 계정 관리: 사용자가 선택한 경우 Apple 또는 Google 계정으로 로그인, 연결, 로그아웃, 계정 삭제 제공

2조. 처리 항목

루틴 정보: 할 일 제목, 예약 날짜와 시각, 반복 요일, 집중 시간, 알림 설정

완료 정보: 완료 또는 못 함 상태, 시작·완료 시각, 선택적으로 촬영한 인증 사진

Screen Time 선택: 사용자가 차단 대상으로 고른 앱, 카테고리, 웹사이트의 시스템 토큰

선택적 계정 정보: Apple 또는 Google이 제공하는 이름, 이메일 주소, 인증 제공자 식별자와 Firebase 사용자 ID

인증 SDK 처리 정보: Firebase Authentication과 Google Sign-In은 개인정보 매니페스트에서 전화번호, 대략적 위치, 사용자·기기 식별자, 기타 사용·진단 데이터와 기타 데이터의 처리를 선언합니다. 실제 처리 범위는 사용자가 선택한 로그인 제공자와 계정 설정, SDK 동작에 따라 달라질 수 있습니다.

인증 보안 정보: Firebase Authentication이 계정 보안과 부정 이용 방지를 위해 처리하는 IP 주소와 사용자 에이전트

지금해 앱은 전화번호, 위치, 연락처 또는 광고 식별자를 직접 요청하거나 자체 서버에 저장하지 않으며, 광고 또는 추적 목적으로 정보를 사용하지 않습니다.

3조. 외부 전송 및 제3자 제공

선택적 계정 연결에는 Google LLC가 제공하는 Firebase Authentication과 Google Sign-In을 이용합니다. 이름, 이메일 주소, 인증 식별자, IP 주소, 사용자 에이전트 및 인증 SDK가 선언한 정보는 로그인, 계정 보안, 부정 이용 방지와 SDK 품질 분석을 위해 미국 소재 시스템에서 처리될 수 있습니다.

이전 시점과 방법은 사용자가 계정 연결 또는 계정 관리 기능을 사용할 때 암호화된 네트워크 통신으로 전송되는 방식이며, Firebase 인증 계정을 삭제할 때까지 보관됩니다.

루틴, 완료 기록, 인증 사진, Screen Time 선택 정보는 Firebase 또는 운영자 서버로 전송하지 않습니다.

광고 SDK와 이용자 추적 기능은 사용하지 않습니다. 인증 SDK가 선언한 품질·진단 분석 외에 루틴 사용 행동을 분석하거나 광고 프로필을 만들지 않습니다.

알람, 실시간 현황, 알림, Screen Time 차단은 iOS가 제공하는 AlarmKit, ActivityKit, UserNotifications, FamilyControls, ManagedSettings 기능을 사용해 기기에서 처리됩니다.

4조. 보유 및 삭제

루틴과 기록은 사용자가 앱에서 삭제하거나 앱을 삭제할 때까지 기기에 보관됩니다.

개별 완료 기록을 삭제하면 연결된 인증 사진도 함께 삭제됩니다.

설정 > 계정 > 계정 삭제에서 본인 확인 후 Firebase 인증 계정을 즉시 삭제할 수 있습니다. Firebase의 백업 시스템에서는 삭제 후 최대 180일 이내 제거될 수 있습니다.

iCloud 기기 백업을 사용하는 경우 iOS의 백업 정책에 따라 앱 데이터가 사용자의 iCloud 백업에 포함될 수 있습니다.

5조. 기기 권한

알람·알림: 예약한 루틴을 정해진 시각에 안내하기 위해 사용합니다.

Screen Time: 사용자가 직접 선택한 방해 앱과 웹사이트를 집중 시간 동안 차단하기 위해 사용합니다.

카메라: 사용자가 인증 사진 촬영을 선택한 경우에만 요청합니다. 사진 없이도 루틴을 완료할 수 있습니다.

권한은 iPhone 설정에서 언제든 변경할 수 있습니다.
